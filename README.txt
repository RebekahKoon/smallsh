## How to Compile
* A makefile is used in order to compile the program.
* Type "make" into the command line (without quotes) to create the smallsh executable
* After the program has compiled, type "./smallsh" into the command line in order to run the program
* In order to run the test script, type "./p3testscript 2>&1" into the command line
* When finished, type "make clean" to remove the .o files or "make cleanall" to remove the
  .o and executable files
